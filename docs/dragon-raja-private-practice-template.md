# Private Practice Template: Dragon Raja / Long Zu-Inspired Workflow

## Important boundary

This file is a template for private learning only. Do not paste copyrighted full text from *Dragon Raja / Long Zu* into this repository.

Use this template with your own short notes outside the repository, or copy the template into a private local document that is not committed.

## Goal of the practice

Practice the workflow of analyzing a complex Chinese fantasy novel and planning a continuation, while keeping the final app designed for original or user-authorized works.

## How to use this template safely

1. Do not paste full chapters.
2. Do not paste long excerpts.
3. Write short personal summaries in your own words.
4. Focus on structure, emotions, conflicts, and continuation planning.
5. Replace any copyrighted details with placeholders before committing anything.
