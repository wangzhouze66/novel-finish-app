# Copyright Boundary

## Purpose

This document defines what is safe to include in this repository and what should remain outside it.

The project is intended to support original or user-authorized novels. A private fan-drafting workflow may be used for learning, but copyrighted full novel text must not be stored in the repository.

## Repository-safe content

The following content is allowed:

- Project plans
- Prompt templates
- Empty templates
- Placeholder text
- Short synthetic examples
- User-created original text
- User-authorized text
- High-level workflow descriptions
- Notes about app behavior

## Content not allowed in the repository

Do not commit:

- Full chapters from copyrighted novels
- Long copied excerpts from copyrighted novels
- Scraped novel text
- Pirated datasets
- Detailed scene-by-scene copied reconstructions intended to replace the original
- Prompt outputs that closely reproduce protected source text

## Private practice boundary

The user may privately practice with their own notes about *Dragon Raja / Long Zu* as a learning exercise.

However:

- Keep copied source text outside the repository.
- Use short personal summaries instead of pasted chapters.
- Do not commit private copyrighted source material.
- Do not build the app around distributing or reproducing protected text.
- Design the final app for original or user-authorized works.

## Good placeholder example

```text
[Placeholder Novel Summary]
A young protagonist enters a hidden academy, discovers a dangerous legacy, forms complicated friendships, and must choose between personal loyalty and a larger conflict.
```

This is acceptable because it is generic and does not reproduce protected expression.

## Bad repository example

```text
[Do not paste a full chapter or long published passage here.]
```

This is not acceptable because the repository should not store copyrighted full text or long excerpts.

## App design recommendation

The app should show a clear notice near input fields:

> Only upload or paste text that you own, created yourself, or are authorized to use. Do not upload copyrighted full novel text without permission.

## Generated output guidance

Generated chapters should be based on:

- The user's original content
- The user's authorized content
- User-provided high-level notes
- Generic writing instructions
- Newly generated original expression

Generated output should not be designed to impersonate, replace, or continue a protected work for public distribution without rights.
