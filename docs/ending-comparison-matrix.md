# Ending Comparison Matrix

## Purpose

The ending comparison matrix helps the lab compare routes before drafting prose. It prevents the system from choosing an ending only because it sounds dramatic in isolation.

Each route should be scored against the same story bible, unresolved thread list, target mixed style, and user preference.

## Scoring scale

Use a 1-10 score for each criterion.

| Score range | Meaning |
| --- | --- |
| 1-3 | Weak fit; likely needs redesign. |
| 4-6 | Usable but has major tradeoffs. |
| 7-8 | Strong candidate with manageable risks. |
| 9-10 | Excellent fit; should be considered for expansion. |

## Comparison criteria

| Criterion | What it measures | Questions to ask |
| --- | --- | --- |
| Completeness | How fully the route resolves promised story material. | Are major conflicts answered? Are important character arcs paid off? |
| Emotional power | Strength and clarity of the reader's emotional experience. | Does the route create catharsis, grief, awe, tenderness, or resonance? |
| Tragedy level | Degree of loss, sacrifice, irreversibility, and fatalism. | Is the tragedy earned? Does it deepen the theme rather than shock the reader? |
| Reader satisfaction | Likely sense of fairness and payoff. | Would readers feel the ending respects their investment? |
| Story logic | Cause-and-effect coherence. | Do events follow from established choices, limits, and consequences? |
| Character consistency | Fit with established motivations and relationships. | Do characters make choices that feel like evolved versions of themselves? |
| Worldbuilding consistency | Fit with rules, factions, history, power systems, and scale. | Does the ending solve the world conflict without breaking the world? |
| Style consistency | Fit with the target mixed style. | Does it balance youth, tragedy, epic scale, restraint, humor, and reflection? |
| Originality | Freshness of the route's structure and emotional payoff. | Does it avoid generic final-battle or sudden-miracle solutions? |
| Expansion potential | Ability to become a final volume, chapter outlines, and scenes. | Is there enough staged progression for long-form development? |

## Weighted recommendation

For the current project direction, recommended default weights are:

| Criterion | Weight |
| --- | --- |
| Story logic | 15% |
| Character consistency | 15% |
| Style consistency | 15% |
| Emotional power | 15% |
| Completeness | 10% |
| Worldbuilding consistency | 10% |
| Reader satisfaction | 10% |
| Originality | 5% |
| Expansion potential | 5% |

Tragedy level is not automatically good or bad. Score it separately as a descriptive intensity, then discuss whether that intensity fits the selected route type.

## Decision outcomes

After scoring, choose one of these outcomes:

1. Expand one route directly into a final volume outline.
2. Revise the strongest route before expansion.
3. Hybridize two or more routes.
4. Generate another route because all current options fail a key criterion.
5. Pause and update the story bible because the route disagreement reveals missing source information.
