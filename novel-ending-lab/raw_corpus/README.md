# raw_corpus 原始语料目录

> 本目录用于存放用户本地的小说原始文件和参考文本。当前第一阶段只服务《龙族》测试小说的私有实验，不生成正文、不公开发布、不商业化。

## 子目录

| 目录 | 用途 |
| --- | --- |
| `target_novel/` | 存放当前目标小说文件，例如《龙族》测试语料。 |
| `author_corpus/` | 存放同作者其他作品或访谈、随笔等本地参考文本。 |
| `literary_references/` | 存放文学参考文本、风格参照、公开领域或已授权资料。 |

## 使用原则

- 原始文本只放在本地，不提交到公开仓库。
- 第一版读取脚本只做文件扫描、大小统计和 `.txt` / `.md` 字符数统计。
- 不在本目录内生成结尾方案、大纲或章节正文。
