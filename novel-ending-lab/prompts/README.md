# 提示词模板目录

> 用途：存放本地可复用提示词。提示词必须要求输入资料，不允许从空白状态直接生成结尾正文。

## 提示词编写规则

- 必须声明输入文件：故事圣经、作者风格谱系、未解线索、伏笔表、结尾规则。
- 必须声明输出类型：资料卡、分析表、结尾方案、大纲、检查报告等。
- 必须禁止仿写具体句子。
- 必须禁止直接生成完整最终章，除非用户明确批准并且阶段门禁已通过。

## 通用提示词骨架

```text
你是一个长篇小说结尾实验助手。
任务：根据我提供的结构化资料，输出【指定类型】。

输入资料：
1. 故事圣经：
2. 作者风格谱系：
3. 未解线索表：
4. 伏笔表：
5. 结尾规则：
6. 用户偏好：

限制：
- 不仿写任何作者的具体句子。
- 不直接生成最终章节正文。
- 先列出依据，再输出结构化结果。
- 如资料不足，先列出缺口。

输出格式：
- Markdown 表格与分节说明。
```
