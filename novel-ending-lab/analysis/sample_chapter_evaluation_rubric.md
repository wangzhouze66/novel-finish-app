# 样章质量评估模板

## 用途

用于评估 DeepSeek / ChatGPT / Claude 等模型生成的《龙族》结尾样章质量。

本模板只用于样章评估与改写决策，不用于直接生成小说正文。评估时应先对照已整理的故事资料、人物卡、世界观规则、未解线索、伏笔表、作者风格笔记与结尾路线目标，再给出评分。

## 评分说明

每项 0-10 分：

- 0-3：严重不合格，存在根本性问题。
- 4-5：勉强可辨认方向，但问题明显。
- 6-7：基本可参考，需要较多修改。
- 8-9：质量较高，只需局部修订。
- 10：非常优秀，可作为标杆样本。

> 注意：第 5、12、13 项是“风险项”。评分时仍按 0-10 分记录，但含义为“控制得越好分数越高”。例如“是否像 AI 机械生成”中，10 分表示几乎不像 AI 机械生成，0 分表示高度机械化。

## 评分维度

| 序号 | 维度 | 分数（0-10） | 评估要点 | 问题记录 / 修改建议 |
| --- | --- | --- | --- | --- |
| 1 | 人物一致性 |  | 角色动机、说话方式、情感反应、关系张力是否符合既有设定；是否避免突然降智、突兀黑化或强行圆满。 |  |
| 2 | 世界观一致性 |  | 龙族血统、学院体系、混血种规则、言灵、龙王、历史与组织逻辑是否自洽；是否避免破坏核心规则。 |  |
| 3 | 原文连续性 |  | 是否承接既有卷章的叙事状态、人物处境、情绪余波和未完成承诺；是否像从前文自然延伸出来。 |  |
| 4 | 伏笔推进质量 |  | 是否推进或回收关键伏笔；是否让伏笔产生新的叙事压力，而不是用解释性台词草草交代。 |  |
| 5 | 新设定风险 |  | 新人物、新组织、新能力、新历史解释是否必要且克制；是否抢走原有主线重量。10 分表示风险很低。 |  |
| 6 | 少年感 |  | 是否保留青春、学院、玩笑、笨拙勇气、朋友间轻微胡闹与失去青春的刺痛感。 |  |
| 7 | 宿命感 |  | 是否体现命运压迫、血统与身份的不可逃避、选择与代价；是否避免空喊“命运”。 |  |
| 8 | 悲剧感 |  | 是否有真实损失、错过、牺牲和余痛；是否避免廉价煽情、为虐而虐或强行刀人。 |  |
| 9 | 群像感 |  | 是否让主要角色、阵营和关系网共同承受结局压力；是否避免所有戏份只服务单一角色。 |  |
| 10 | 语言自然度 |  | 中文表达是否流畅、有节奏、有画面感；对白是否自然；是否避免翻译腔、模板句和堆砌形容词。 |  |
| 11 | 情绪克制度 |  | 情绪是否通过动作、场景、留白与细节呈现；是否避免过度哭喊、反复解释心情或直接灌输感动。 |  |
| 12 | 是否像 AI 机械生成 |  | 是否存在套路化转折、平均用力、重复句式、总结腔、过度工整段落。10 分表示几乎没有机械感。 |  |
| 13 | 是否过度解释设定 |  | 是否避免大段百科式说明；世界观信息是否嵌入行动、冲突和人物认知。10 分表示解释量控制优秀。 |  |
| 14 | 是否适合作为结局开端 |  | 是否具备结局篇章的启动感、危机感、人物再聚焦和路线承诺；是否能自然引出后续大纲。 |  |
| 15 | 是否值得进入下一轮改写 |  | 综合判断样章是否有继续投入价值；是否具备可保留的结构、场景、人物瞬间或情绪核心。 |  |

## 总分标准

| 总分 | 结论 | 处理建议 |
| --- | --- | --- |
| 90-100 | 可进入正文候选 | 保留为候选样章，进行局部润色、连续性核查和风格微调。 |
| 80-89 | 需要局部修改 | 保留主要结构，重点修正人物、设定、节奏或语言问题。 |
| 70-79 | 只能作为思路参考 | 提取可用桥段、伏笔处理或情绪方向，不建议直接改正文。 |
| 60-69 | 结构可借鉴，正文不可用 | 只保留宏观结构或少量设定思路，需要重新生成或重写。 |
| 60以下 | 废稿 | 不进入改写流程，仅记录失败原因以优化提示词。 |
