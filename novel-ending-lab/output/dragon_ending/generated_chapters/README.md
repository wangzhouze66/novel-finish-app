# 样章版本输出目录

这里用于保存不同模型生成的《龙族》结尾样章版本，便于后续使用样章质量评估模板进行横向比较。

> 当前目录只是样章版本的存放位置说明。不要在未完成资料核查、路线确认和用户明确授权前生成小说正文。

## 可保存的文件示例

- `chapter_001_deepseek_expert.md`
- `chapter_001_deepseek_normal.md`
- `chapter_001_gpt.md`
- `chapter_001_claude.md`

## 使用建议

1. 每个文件只保存一个模型、一个提示词版本、一个章节位置的样章。
2. 文件名应包含章节编号、模型名称和生成模式，避免混淆。
3. 每个样章都应回填到 `novel-ending-lab/analysis/sample_chapter_evaluation_rubric.md` 进行评分。
4. 评分后再决定是否进入下一轮改写，不要直接把初稿当作最终正文。
5. 如需比较多个模型版本，应优先比较人物一致性、世界观一致性、原文连续性、情绪克制度和 AI 机械感。
