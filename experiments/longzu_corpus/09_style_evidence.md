# Style Evidence

## Purpose

Collect concrete style observations that can become rewrite rules for Chinese-language prompt outputs and later scene-level expansion.

## Style Evidence Template

| observed feature | source context | sentence rhythm | dialogue rhythm | humor pattern | tragedy pattern | imagery | emotional restraint | scene construction | rewrite rule |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |  |  |  |

## Review Questions

- What is the rhythm before a joke, confession, reveal, sacrifice, or silence?
- How does the prose imply grief without over-explaining it?
- Which images recur enough to become part of the style bible?
- What rule would help a future draft sound closer without copying source prose?
