# Worldbuilding Evidence

## Purpose

Collect rules, concepts, factions, mythology, powers, history, constraints, contradictions, and gaps before building epic-scale ending routes.

## Worldbuilding Evidence Template

| rule or concept | source context | explanation | related characters | related factions | contradictions or gaps | ending implication |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |

## Review Questions

- Is this a hard rule, soft pattern, rumor, myth, or character belief?
- What would break if an ending ignored this rule?
- Does the rule create a cost, limit, loophole, or sacrifice mechanism?
- Which contradictions can become route-level mysteries or reveals?
