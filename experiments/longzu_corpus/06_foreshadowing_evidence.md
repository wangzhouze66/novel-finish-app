# Foreshadowing Evidence

## Purpose

Track setup/payoff material so ending routes can recover promises instead of inventing resolutions from scratch.

## Foreshadowing Evidence Template

| foreshadowing item | source context | related character | possible meaning | payoff possibility | priority | route relevance |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  | Must pay off / should pay off / optional / symbolic | Route A / Route B / Route C / Route D / Route E / hybrid / unknown |

## Review Questions

- Is the setup plot-critical, emotional, symbolic, comedic, or atmospheric?
- Has the source already paid it off, partially paid it off, or left it open?
- What kind of ending would make the payoff feel earned?
- What is the risk if the ending ignores this item?
