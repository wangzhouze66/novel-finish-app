# Agent Instructions

These instructions apply to the entire repository.

## Project intent

This repository is for a Chinese novel continuation MVP. The final app should support original or user-authorized novels only.

## Copyright safety rules

- Do not commit copyrighted full novel text.
- Do not add long excerpts from published novels.
- Do not add scraped novel datasets.
- Do not reconstruct protected chapters in a way that could substitute for the original.
- Use placeholder text, synthetic examples, user-created examples, or user-authorized examples.
- The private *Dragon Raja / Long Zu* practice workflow must remain a template only; do not add actual copyrighted source text from the novel.

## Development approach

- Keep early changes beginner-friendly and well documented.
- Prefer small, clear documents and templates before adding application code.
- Keep prompts reusable and explicit about requiring user-owned or user-authorized input.
- If code is added later, separate app code, prompt assets, generated user data, and tests.

## Writing style

- Use clear Markdown headings.
- Explain why each step exists.
- Include Chinese-language prompt outputs where the app is expected to generate Chinese prose.
